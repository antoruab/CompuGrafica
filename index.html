Course description

The Computer Graphics for Multimedia Engineers course provides training in techniques and tools for the creation and manipulation of computer graphics.

Students will develop key skills for creating interactive web applications and integrating animations, simulations, and virtual and augmented reality projects, applicable in multimedia contexts. In addition, it focuses on strengthening mathematical knowledge, especially in vector geometry, to solve graphic challenges such as surfaces, lighting, and textures.
